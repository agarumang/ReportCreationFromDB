# ReportCreationFromDB

WPF Report generator (target: .NET Framework 4.8).  
Contains code to enumerate DB tables, fetch data and export to Excel/CSV/PDF.

Instructions:
- Open solution in Visual Studio (target .NET Framework 4.8).
- Restore NuGet packages.
- Build and run.
